This is the API

chat.get

URL

https://localhost/api/v1/chat

HTTP Method

GET

Response Type

List

Parameters

  • Query: count BigDecimal

    Number of results to fetch.

  • Query: start BigDecimal

    Starting ID for results.

  • Query: reverse Boolean

    If true, will sort results newest first.

  • Query: channelID Double

    Channel id. GET /chat/channels for ids. Leave blank for all.
